Java Escape Room

Project: Chapter 1 PLTW Problem
Course: AP Computer Science Applications
Duration: Sept 3 - Sept 13 2023
Objective: Create a escape room game with Java where the user can control the object with user-inputted commands
Type of Project: Group Project

-

In this initiative, we aimed to construct an obstacle course utilizing Java, collaborating as a team of four. The functionality allowed users to maneuver the character in any desired direction based on keystrokes, enabling interactions such as picking up items on-screen, restarting the game, and more. Featured below is a bear character, which players could guide across the screen to collect berries and navigate through obstacles. This project was developed on Replit using Java, and my contribution included writing three crucial if statements. Through this endeavor, I gained insights into Java game development and the effective application of classes in programming. A significant challenge we encountered was adapting to Java, a programming language new to us, with particular difficulties in class implementation due to unfamiliarity. Nonetheless, the project provided a valuable learning experience, offering a platform for experimentation and skill enhancement in Java, laying a strong foundation for future projects. The project proved to be a rewarding starting point for the course.
